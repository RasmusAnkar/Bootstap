HTML Element

HTML element är exempelvis html, body och h1 element där de innehåller en start tagg sedan någon innehåll och en sluttagg
h1 eller heading 1 bestämmer prioritering på d olika heading de vill säga att h1 har högre prioritering än h6
body elementet används för att visa vad som är själva dokumentets kropp och innom body elementet är även vart heading och blider ska vara

HTML (generell)

HTML (HyperText Markup Language) är ett kodspråk som används vill tillverkning och ändring av webbsidor
dagens hemmsidor använder HTML 5 eftersom att det är den senaste standarden.
html är även dagens standard för alla hemsidor på World Wide Web

ID

ID eller identifier är ett sätt att ge en element en unik markör som kan användas vid länkning
Ett ID måste vara unikt för varje HTML sida vilket innebär att exempelvis ID "Homepage" endast får stå som ID för ett element
ID är ganska likt Classer men det som skiljer dem åt är att ID bara får använas för ett element medans classer får användas flera gånger

Class

Classer används likt ID för att ge element identifiers vilket är bra för länkning, referering eller att ge elementet egenskaper
det som skiljer Classer från ID är att de får användas på flera element medans ID bara får användas på ett element.
Classer can bland annat med hjälp av CSS användas för att ge Element egenskaper som färg, storelk eller form

Meta

Ordet Meta ursprungar från greekiskan och betyder att referare till sig själv

Responsive

En Responsiv hemmsida innebär att hemmsidan anpassar sig till användarens efter användarens platform
¨ detta är speciellt bra om man har en hemsida som ska kunnas användas av mobilanvändare
detta kan göras genom att basera positionering med hjälp av justify-content classen istället för att definera storlek i form av pixlar

Http

Http eller Hypertext Transfer Protocol är ett kommunikationsprotokoll som används för att överföra data på WWW eller World Wide Webb
Det skapade ursprungligen för att överföra HTML sidor fron Webbservrar till webbklienter

Https

HTTPS eller Hypertext Transfer Protocol Secure är ett protoll för kryptering av webbdata
Detta används på samma sett som HTTP fast med kryptering för att skydda datan som överförs
Fastän krypteringen är datan dock inte helt säker och därför välkommnar jag vår stolta sponsor: NORD VPN

DNS

Domännamnssystemet eller DNS är ett system som förenklar ip-adresseringen på ett nätverk och kopplar ihop domännamn med ip-adresser
Utan en domän kan inte en Webbsida Kopplas samman med WWW eller liknande nätverk

CDN

CDN eller (Content delivery network) är ett innehållsleveransnätverk
Det används för att nätverk som inte är lokaliserade nära varande ska kunna kommunicera via proxy-servrar
Detta ökar driftsäkerhet och ger en bättre prestanda

Compressed data

Compressed data eller datakompression innebär att man omkodar data till fler rader
Fördelen med datakompression är att det blir färre infromationsbärande rader kod vilket i sin tur minskar hur mycket lagringsutrymme den tar
Detta används ofta blandannat i .Zip filer

Local bootstrap

Local bootstrap innebär att en bootstrap fil laddas ner för att användas som en lokal css
Detta är att föredra eftersom datorn inte behöver ta informationen från en online biblotek konstant

JQuery

jQuery är ett javascript biblotek
jQuery är både snabbare och mer simpel än normal javascript och funkar på många olika webbläsare
jQuery är lättare att använda vid animering och bättre vid händelsehantering

Flexbox

Flexbox är ett css biblotek och en layout model som fokuserar på plasering, storlek, form och färg
Flexbox gör det därför lättare att designa flexibla och responsiva layouts eftersom stor del av flexbox bibloteket är skrivet i %
Flexbox är känt för sina Containers vilket ofta används om man ska ha fler små objekt ordnade

Grid

Grid är ett rutnät som används i grafisk design för att strukturera innehåll
Rutnäten är som ett kordinatsystem med en X och Y axel som används för att ge innehållet in position
Pågrund av grids kan man exempelvis mäta avstånd mellan två objekt i grafisk programmering

Code Snippets

Code Snippets är små delar av kod som kan återanvändas och sparas i filer
HTML snippets är exempelvis små block av source code som kan användas gång på gång
Detta kan vara meta taggen eller annat som används i nästan alla projekt

WASM

WASM eller WebAssembly är ett binärt och portablet kodformat
WASM används för att skapa high performance applikationer på webb sidor
WASM är tillgängligt för nästan alla programmeringspråk och operativsystem

Chromium

Chromium är en en webbläsare som är basead på öppen källkod
Många av världens main-stream webbläsare som exempelvis google chorme, microsoft edge och torch är baserade på chromium
Chromium är nästan fungerar nästan identiskt till google chrome fast skiljer sig i att användasspårning inte utförs

Svg

Svg eller Scalable Vector Graphics är ett vektorgrafik-format för tvådimensionella bilder
Svg stödjer sedan animatioer och interaktivitet så bilder exempelvis kan röra på sig, vridas och vändas
Svg är en öppen standard och släpptes 1999

png-24

png eller portable network graphic är en typ av fil eller bildformat
Numret efter beskriver hur mång bit färger som filen stödjer så png-24 stödjer 24-bit färg
png-24 är dagens standar och har tillgång till 16 miljoner olika färger
